# 项目设定总结

## 1. 项目愿景
本项目是一款以科幻世界观为背景的在线 FPS 刷宝游戏，采用赛季制运营。
游戏强调射击手感、装备成长、关卡挑战和持续更新内容，让玩家在长期游玩中不断获得新的探索目标与战斗动力。

## 2. 核心定位
本作的核心定位是“科幻题材的网络刷宝射击游戏”。
它融合了快速直接的 FPS 战斗体验与长期养成、重复挑战、装备收集和内容迭代的网游结构。

## 3. 世界观概述
在遥远的未来，人类向未知前线区域持续扩张，并在各类危险地带中发现了失落技术、异常区域、废弃设施与高价值资源。
这些区域并不稳定，且会随着时间推移发生变化，因此每个赛季都可以作为一次新的世界事件，带来新的地图、副本、敌人与剧情线索。

## 4. 玩家身份
玩家扮演前线探险者、回收人员或合同制行动员。
他们进入高危区域执行任务，回收资源、争夺装备、调查异常，并在不断升级的敌对环境中生存下来。

## 5. 核心玩法循环
游戏的核心循环应围绕“进入、战斗、获取、强化、再挑战”展开。
典型流程如下：
进入副本或地图 -> 清理敌人 -> 获取装备与材料 -> 完成目标 -> 返回结算 -> 强化角色与武器 -> 挑战更高难度内容。

## 6. 赛季制理念
本项目采用赛季制内容更新。
每个赛季都会推出新的装备、地图、副本、Boss、敌人类型和剧情碎片，让游戏世界持续扩展。
赛季不只是补丁，而是世界事件的一部分，是推动玩家回归和持续成长的主要动力。

## 7. 内容更新方向
每个赛季的更新重点包括但不限于以下内容：
- 新装备
- 新地图
- 新副本
- 新敌人
- 新 Boss
- 新剧情碎片
- 新玩法机制

## 8. 风格与氛围
整体风格应保持科幻、未知、前线、紧张和具有探索感。
氛围上既要有冒险与发现的兴奋感，也要保留高风险环境带来的压迫感与危机感。
后续可以进一步确定是偏硬科幻、失落文明、商业殖民，还是异常灾变风格。

## 9. 主要待定事项
以下内容需要在后续继续明确：
- 单局结构是副本制还是开放区制
- 职业系统是固定职业还是装备流派驱动
- 赛季长度是多少
- 旧赛季内容是否保留
- 故事是主线推进还是事件驱动
- 战斗节奏更偏硬核还是更偏爽快
